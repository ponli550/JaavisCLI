# One-Army Architecture Template

This is the standard architecture template for "One-Army" projects.

## 1. Stack Choices
*   **Web**: React/Next.js/Vite
*   **Mobile**: React Native (Expo)
*   **Backend**: Node/Python/Go
*   **Database**: Postgres/Supabase

## 2. Directory Structure
*   `apps/`
*   `packages/`
*   `services/`
